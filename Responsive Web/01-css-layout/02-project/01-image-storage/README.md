## Specs
Kamu adalah seorang photographer yang aktif, kamu ingin membuat penyimpanan gambar yang responsive untuk mempermudah kamu dan orang lain dalam mencari gambar tersebut

- Gunakan template yang sudah disediakan pada file `index.html` dan `style.css`. **Jangan melakukan perubahan pada file html**
- Kita akan membuat web responsive dengan `flexbox`, `grid`, dan `@media`
- Gunakan display `flex` pada class `container` untuk membuat kedua nya sejajar secara horizontal
- Buatlah code kamu di style.css
- Buatlah `display: grid` untuk class `image` 
- Gunakan `grid-template-columns` pada class `image` untuk mengatur kolom tiap item dalam class tersebut 
- Buatlah media query untuk breakpoint pada ukuran minimal 700px ke atas
- Ubah display `flex` pada class `container` menjadi `grid`
